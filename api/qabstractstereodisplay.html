    Detailed Description

    The QAbstractStereoDisplay class is the base class for all stereoscopic display device implementations.

    Member Function Documentation

    float QAbstractStereoDisplay::aspectRatio() const

    Returns the display's aspect ratio, i.e. the ratio of horizontal to vertical resolution.

    void QAbstractStereoDisplay::enableEyeTracking(const bool enable) [pure virtual]

    If enable is set to true then eye tracking is enabled, otherwise it is disabled.

    void QAbstractStereoDisplay::enableOrientationTracking(const bool enable) [pure virtual]

    If enable is set to true then orientation tracking is enabled, otherwise it is disabled.

    If the device does not support orientation tracking, then this member function does nothing.

    void QAbstractStereoDisplay::enablePositionalTracking(const bool enable) [pure virtual]

    If enable is set to true then positional tracking is enabled, otherwise it is disabled.

    void QAbstractStereoDisplay::enableVsync(const bool enable) [pure virtual]

    If enable is set to true, then vertical synchronization (VSYNC) is enabled.

    QPointF QAbstractStereoDisplay::eyePosition() const [pure virtual]

    Returns the normalized screen-space eye position.

    QSize QAbstractStereoDisplay::eyeResolution() const

    Returns the display's eye resolution, i.e. half of the total resolution.

    bool QAbstractStereoDisplay::eyeTrackingAvailable() const [pure virtual]

    Returns true if eye tracking is available, false otherwise.

    bool QAbstractStereoDisplay::eyeTrackingEnabled() const [pure virtual]

    Returns true if eye tracking is enabled, false otherwise.

    QQuaternion QAbstractStereoDisplay::headOrientation() const [pure virtual]

    Returns the display device's orientation quaternion.

    QVector3D QAbstractStereoDisplay::headPosition() const [pure virtual]

    Returns the display device's head position. If positional tracking is disabled, then QVector3D(0, 0, 0) is returned.
