    bool QStereoEyeCamera::projectionChanged() const

    Returns true if the camera's projection parameters have changed, false otherwise. If any projection parameters have changed, then the camera's projection matrix must be recalculated before it can be used.

    See also setProjectionChanged().

    void QStereoEyeCamera::setProjectionChanged(const bool changed)

    If changed is set to true, then the projection matrix needs to be recalculated before it can be used.
